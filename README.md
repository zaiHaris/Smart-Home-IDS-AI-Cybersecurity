# 🛡️ AI-Powered Smart Home Intrusion Detection System

## Overview

This project presents an AI-driven Intrusion Detection System (IDS) for smart home IoT environments. The system monitors IoT device activity, detects anomalous behavior using Machine Learning, and provides real-time cybersecurity monitoring through an interactive SOC-style dashboard.

---

## Features

✅ AI-Based Anomaly Detection (Isolation Forest)

✅ Real-Time Dashboard (Flask + SocketIO)

✅ IoT Device Activity Simulation

✅ Threat Severity Classification

✅ Incident Response Workflow

✅ Security Analytics Dashboard

✅ WhatsApp Critical Alert Notifications

✅ Live WebSocket Updates
